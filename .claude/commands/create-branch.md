# Branch Creation Command

Create and switch to a branch.

## Instructions

$ARGUMENTS

## Execution Steps

Follow this systematic approach:

1. Understand the Context
    - Read `docs/git-guidelines.md` to understand the project's Git conventions
    - Understand the instructions (if any)

2. Create New Branch
    - Provide a branch name following project conventions
    - Create and switch to the branch: `git switch --create "Your branch name"`

## Output

Include the following in the output:

- **Summary**
    - Created branch name

## Important Notes

- Always use English for branch names
- Do NOT add or modify any code
